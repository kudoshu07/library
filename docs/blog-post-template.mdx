<h2>メモ（運用ルール）</h2>
<ul>
  <li><code>slug</code> はURLになるので英数字・ハイフン推奨</li>
  <li><code>tags</code> は <code>#</code> なしで記載（例: <code>startup</code>）</li>
  <li><code>thumbnail</code> の画像ファイル名はASCII推奨（日本語ファイル名は本番で壊れやすい）</li>
  <li>保存先は <code>content/blog/YYYY/MM/DD/slug.mdx</code></li>
  <li>公開URLは <code>/YYYY/MM/DD/slug/</code></li>
</ul>
