# Drafts (Non-Public)

This folder is for writing blog drafts.

- Files in `content/drafts/` are **not** loaded by the blog content loader.
- Public blog posts must be placed under `content/blog/YYYY/MM/DD/slug.mdx`.

## Suggested workflow

1. Create or edit a draft in this folder.
2. When ready to publish, move it to `content/blog/YYYY/MM/DD/`.
3. Update frontmatter (`title`, `date`, `slug`, etc.) before publish.
