<p>10代で起業して、右も左も分からない大学生だった僕が今までなんとか会社を継続できているのは、先人の知恵をお借りさせていただいたからだと思っています。</p>

<p>（最近は起業当初ほど、社外の先輩方に相談させていただくことが多く、その度に優しく対応してくださって本当に感謝してもし尽くせない...）</p>

<blockquote class="wp-block-quote">
<p>自分に知恵がないならば、恥を忍んででも借りなければいけない。そうしなければ、40,50代の経験・資金・人脈を豊富に持ち合わせた大人たちと対等に戦うことなんてできない。</p>

<h2 class="wp-block-heading">３つの相談フェーズ</h2>

<p>一言で「相談する」と言っても、３つの段階があると思います。</p>

<p>簡単にまとめると、以下のイメージです。</p>

<figure class="wp-block-table is-style-regular"><table><thead><tr><th>段階</th><th>状態</th><th>具体例</th></tr></thead><tbody><tr><td><strong>1. ちんぷんかんぷんフェーズ</strong></td><td>・前提知識がない<br>・調べても分かんない<br>・何を聞けば良いかもわかんない</td><td>シリーズAの資金調達について<br>・企業価値や判断材料などが不明<br>・調べても数記事だけ。詳細は不明</td></tr><tr><td><strong>2. 方針決定フェーズ</strong></td><td>・方針を決定する時期<br>・そのための知識、経験不足<br>・具体的な事例が知りたい</td><td>・資金調達をするかしないか<br>・エクイティかデッドでするか<br>・個人投資家に多く参加してもらうか</td></tr><tr><td><strong>3. アウトプットフェーズ</strong></td><td>・方針は決定している<br>・アウトプットに落とす段階</td><td>・具体的な企業価値の金額<br>・ピッチ資料の内容 など</td></tr></tbody></table></figure>

<p>それぞれのフェーズに合わせて、相談先・内容・スタンスなどを分けるのが良いと思っています。</p>

<p>特に、1. ちんぷんかんぷんフェーズで相談することは、「こんな状態で聞かせてもらっても良いのかな...」と気が引けますが、逆にこのタイミングで相談できると、数ヶ月間のタイムロスを防ぐことができます。（だから頑張るべき）</p>

<h3 class="wp-block-heading">1. ちんぷんかんぷんフェーズ</h3>

<p>「〇〇しないと行けないけど、何にもわかんないし、調べても全然出てこない」というフェーズ。</p>

<p><strong>理想的な流れ</strong></p>

<p>まず「全体像・相場感」が必要になるので、以下の流れが理想的だと思います。</p>

<ol>
<li>ググって記事を読み漁る。できればササっと本も読む。</li>

<li>専門家・経験者にラフに相談させていただき、得た知識が正しいか確認。</li>
</ol>

<p>これだけでちんぷんかんぷんフェーズはクリアできます。</p>

<p><strong>気をつけること</strong></p>

<ol>
<li>知識不足を恥じない。正直に「わからない」と伝え、教えを乞う。</li>

<li>最低2人に相談する。偏った情報に盲信的にならないよう、複数人に相談し相場感をつかむ。</li>
</ol>

<p><strong>相談する相手</strong></p>

<ul>
<li>取り組むイシューの経験者・専門家</li>

<li>第一線で活躍する先輩</li>

<li>非専門家のフワッとした助言に注意</li>
</ul>

<p><strong>理想の例</strong></p>

<ul>
<li>「この本読むと良いよ」「この記事良いよ」が出る</li>
</ul>

<h3 class="wp-block-heading">2. 方針決定フェーズ</h3>

<p>「全体像を掴み、大きな方針を決定するための知識と経験がない」というフェーズ。</p>

<ul>
<li>全体像を掴んだ上で、自分がAかBどちらかの選択を迫られるシーン。</li>

<li>具体的には、資金調達をエクイティかデッドでするか、広告を代行するか自社でやるか等</li>
</ul>

<p><strong>理想的な流れ</strong></p>

<p>このフェーズでは、「事例とその背景」が必要になるため、以下の流れが理想だと思います。</p>

<ol>
<li>noteやインタビュー記事を探して、方針決定の事例とその背景を知る。</li>

<li>その上で、自ら方針の仮説を設定する。</li>

<li>方針決定経験者に相談し「なぜAを選び、Bを選ばなかったか？」を質問、また自分の仮説についてのフィードバックをもらう。</li>
</ol>

<p>これを繰り返し、方針決定フェーズをクリア（= 方針を決定）する。</p>

<p><strong>気をつけること</strong></p>

<ol>
<li>仮説を持って相談に臨む。仮説がなければ、良いフィードバックは得られない。未完成でも仮説をぶつけて、それに厳しいフィードバックをいただく。</li>

<li>正解はない。だからこそ、失敗確率を減らすために相談する。</li>

<li>方針は長期的なもの。相談した時に、うまくいっていた方針も、半年後には失敗しているかもしれない。</li>
</ol>

<p><strong>相談する対象</strong></p>

<ul>
<li>最近、方針決定した人</li>

<li>方針決定して、成功した人</li>

<li>方針決定して、失敗した人 = 自分の一歩先を行く人</li>
</ul>

<h3 class="wp-block-heading">3. アウトプットフェーズ</h3>

<p>「具体的なアウトプットのフィードバックを集める」フェーズ。</p>

<p>方針決定したため、後はアウトプットするのみだけど、経験不足ならフィードバックは必須です。</p>

<p>例えば、評価制度の詳細や企業価値の具体的な数字などは、ググっても出てきにくい。</p>

<p><strong>理想的な流れ</strong></p>

<p>このフェーズでは、「具体例」が必要になるため、以下の流れが理想的だと思います。</p>

<ol>
<li>noteやSNSを漁ってリアルな情報収集（このフェーズだと、知り合いに聞けることも多い）</li>

<li>その上で、一度アウトプットしてみる。</li>

<li>アウトプットしたことがある人/アウトプットに触れたことがある人に相談し、フィードバックをもらう。</li>

<li>それを元に、改善する。</li>
</ol>

<p>これを繰り返し、最終的なアウトプットを完成させる。</p>

<p><strong>気をつけること</strong></p>

<ol>
<li>自分のアウトプットを持って相談に臨む。事前にシェアしておき、目を通しておいてもらえると最高。</li>

<li>どの観点でフィードバック欲しいか伝える。これがないと、「自分でもわかっているのに...」なフィードバックが多くなり、お互いに不毛な時間になってしまう。</li>
</ol>

<p><strong>相談する対象</strong></p>

<ul>
<li>アウトプットしたことがある人。</li>

<li>アウトプットに触れたことがある人
<ul>
<li>例えば、評価制度であれば、アウトプットフェーズまで来ると、「評価制度を作った人」だけではなく「評価される人」も相談対象となり、対象が広がる。</li>
</ul>
</li>
</ul>

<h2 class="wp-block-heading">無知の知</h2>

<p>ほとんど昔に社内notionに書いたドキュメントを再度整理するようなブログになりましたが、自分で改めて振り返ることができてよかったです。</p>

<p>ここでの本質は「無知の知」であり、知らないことを恥じず、間違っているかもしれない仮説をぶつける勇気を持つことだと思います。</p>

<p>世の中知らないことだらけなので、これからもたくさんの人に相談させていただきながら、できる限り自分も相談に乗ることができる人生にしていきたいです。</p>

<p>普段相談させていただいている、本当に優しい先輩方に心からの大感謝を。</p>
