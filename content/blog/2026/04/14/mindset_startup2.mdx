<p>以前、<a href="https://kudoshu07.com/2023/07/07/mindset_startup">「スタートアップ」として戦うためのマインドセット（自戒）</a>を書いてからもう3年が経とうとしている。</p>

<p>当時から、会社の状況・個人の考え・事業フェーズも変化した。</p>

<p>このブログでは2026年4月現在の状況に合わせて、「スタートアップ」「ヴィーガン業界」「飲食事業」の3つに分解して改めて自戒として残しておきたい。</p>

<h2>はじめに：読んでくださる方へ</h2>

<h3>目的じゃないこと</h3>

<ul>
<li>このブログでは、<strong>僕個人が8年間の社会起業家経験で学んできたこと・反省してきたこと</strong>を言語化しています。</li>
<li>そのため、全てを100%完璧にできることを<strong>全員に求めるものではありません</strong>。</li>
<li><strong>「こうじゃないといけない」と自己否定させることは目的ではない</strong>です。</li>
</ul>

<h3>目的</h3>

<ul>
<li>目的は、意思決定に<strong>迷ったときなどに立ち返るオリジン・悩んでる人の参考になること</strong>です。</li>
<li>迷った時に、<strong>補助線として参考にしてもらうこと</strong>があれば嬉しいです。</li>
</ul>

<h2>“スタートアップ”として戦う</h2>

<h3>Startup = Growth</h3>

<ul>
<li><strong>スタートアップとは、短期間で急成長する企業</strong>であり、それゆえに投資家から資金が集まる。その成長機会を求めて人が集まる。集まったリソースを使い、事業を加速度的に成長させ、その実績がまたリソースを集める。</li>
<li><strong>だからこそ、既存の企業にはできないチャレンジができる。社会を早く・大きく変えることができる。スタートアップの本質は”急成長”にある。</strong>また、VC(ベンチャーキャピタル)などとの投資契約には「上場or売却」を一定期間内にする努力義務が含まれる。</li>
<li>僕たちは、「誰もがヴィーガンを選択できる”Hello Vegan!”な社会をつくる」<strong>Missionを達成するために、この戦い方を主体的に選択している</strong>。そのため、あらゆる業務にも<strong>「急成長につながるか？」「急成長に耐え得るか？」</strong>という観点を持って臨む。</li>
</ul>

<p><em>参考：<a href="https://www.paulgraham.com/growth.html">Startup = Growth</a></em></p>

<h3>「難しい」から意味がある</h3>

<ul>
<li>家族や友人はもちろん、投資家や他経営者にも<strong>「難しい」と指摘されることが多い</strong>。その言葉通り、<strong>基本的にほとんどのことは上手くいかない</strong>。嬉しいことよりキツいことの方が多い。</li>
<li>逆説的だが、<strong>だからこそやる意味がある</strong>。</li>
<li>誰もが「できる」「簡単」と思うことを、僕たちがやる必要はない。むしろ、<strong>他の人が不可能と思っているが、やる意義があること。あるいは僕たちだからできることにフォーカスする</strong>。</li>
<li>そのため、困難に遭遇する可能性は100%だ。それでも粘り強く、成功するまで諦めずにやる。弛まぬ努力と、絶対に成し遂げる覚悟でやる。</li>
<li>つまりは<strong>Plus Ultra</strong>であり、<strong>”良い受難を!!”</strong>ということ。</li>
</ul>

<h3>完璧主義を捨て、早く世に出そう</h3>

<ul>
<li>急成長させるためには、とにかく<strong>リリース速度が重要</strong>だ。<strong>完全に知識を集め、完璧なアウトプットをつくってから世に出しては遅すぎる</strong>。</li>
<li>とにかく早く世に出そう。ただし、もちろんクオリティの低いものではダメだ。重要なのは<strong>「最小限でリリースする」</strong>こと。結果、短期間でフィードバック回収・改善を繰り返すことで、最高速度でブチ抜くことができる。</li>
<li>頭の中だけで検証しない。パソコンに向かって時間を浪費しない。100%を目指して、お客様にとって価値のない要素に時間をかけない。<strong>完璧主義を捨てよう。</strong>恥ずかしいくらいがちょうど良い。とにかく建物から出て、パソコンの前から離れて、世に出して検証しよう。</li>
<li><strong>実行して、実行して、実行しよう。</strong>目標達成に強い意識を持とう。頭でっかちに計画や制度をつくるのに時間をかけ過ぎず、手足をつかって目標数値を1でも増やそう。<strong>毎日、毎週の目標達成の数%の積み重ねが数年後に数百倍となって現れる</strong>。毎日、毎週、しっかり必死にやり切る。</li>
</ul>

<p><em>参考：<a href="https://kudoshu07.com/2026/01/04/execute2025">2025：実行 (振り返り)</a></em></p>

<h3>つくってこわそう！</h3>

<ul>
<li>一度目のアイデアで成功するならそんなにラッキーなことはない。けれど、ほとんどの場合は<strong>いくつもアイデアを出し、その中から良いものを何度もつくり、それでも上手くいかないこと(あるいは中途半端に上手くいくこと)がほとんど。</strong></li>
<li><strong>いっぱいつくって、いっぱいこわそう。</strong>みんなで頑張ってつくったもの、一度世に出したものを壊すのは怖い。しかし、<strong>それを壊さずに放置し、じわじわと死んでいくのはもっと怖い。</strong></li>
<li>自己否定と自己研鑽を常に繰り返すことで、適切な進化を遂げることができる。「本当にこれで良いか？」「今のままで目的を達成できるか？」と上手くいっている時ほど疑おう。</li>
<li>自分個人の<strong>働き方やスタンス、思想も、どんどんつくってこわそう</strong>。事業は毎年毎年大きく変化する。その変化に、個人の成長も伴わせる必要がある。変化を恐れていては、世界は変えられない。</li>
</ul>

<h3>インサイトからはじめる</h3>

<ul>
<li>小さい企業に「なんか良さそうなアイデア」にかける時間もお金もない。それは資金と人を持て余した大企業のやり方だ。リソースの乏しい僕たちは、<strong>失敗する可能性を極限まで下げる</strong>ことが求められる。</li>
<li>だから、インサイトからはじめる。<strong>力のないスタートアップの勝ち筋は「競合が知り得ない深くて広いインサイトを元に事業開発をすること」</strong>だと考える。誰よりもお客様のことを知ろう。<strong>お客様よりも詳しくなろう。</strong></li>
<li>求められるものを作ろう。ただし<strong>、「BではなくAが欲しい」と言われてAをつくるのはナンセンス</strong>だ。もっと深く「〇〇の状況にある場合、Bでは〇〇に不満があるため、〇〇を求めてAがほしいと感じている」というように、文脈を理解し、真に求められるCを提供しよう。</li>
</ul>

<p><em>参考：<a href="https://kudoshu07.com/2024/07/14/insightdriven">インサイトからはじめよ（Insight Driven）</a></em></p>

<h3>恥を捨て、先人の知恵を借りる</h3>

<ul>
<li>自分の知識不足、経験不足、センス不足を認めよう。目的達成のための正しい努力は、自分の弱さを受け入れることから始まる。しかし、その弱さは諦める理由にはならない。<strong>正しい現在地を知ることは、より早く目標地点に辿り着くための一歩目だ</strong>。</li>
<li><strong>恥を捨てて、先人の知恵を借りよう。</strong></li>
<li>不足しているものを１から自分で獲得していては時間がかかり過ぎてしまう。<strong>年下だろうが、初対面だろうが、頭を下げて「教えてください」と伝えよう。</strong></li>
<li><em>「俺は、俺が弱いことをとうの昔に知っている。」</em>星海光来（ハイキュー!!）</li>
</ul>

<p>参考：<a href="https://kudoshu07.com/2023/04/28/chiekari">若輩者は「先人の知恵を借りる」でなんとかする</a></p>

<h3>最優先事項を優先する</h3>

<ul>
<li>人は得てして、<strong>「重要なこと」ではなく、「緊急のこと」を優先してしまいがち</strong>だ。しかし、それでは「なんだか毎日忙しいけど、事業は成長していない」という最悪の事態になる。<strong>せっかく大切な人生を使い、努力を惜しまず注いでいるのに、成果が出ない悲しいことになる</strong>。</li>
<li><strong>不完全であることは仕方がない。</strong>リソースの乏しい零細企業であることを受け入れよう。</li>
<li>その上で、<strong>最も注力すべき重要事項を見極め、一点突破でリソースをそこに集中投下しよう。</strong>そのためには、最重要事項を見極める能力が求められる。そして、それを優先する(=他を優先しない)勇気が必要だ。</li>
<li><em>“一心不乱に大量の仕事をすることでバリューをあげようとしてもムダ！” 『イシューからはじめよ』</em></li>
</ul>

<h3>10倍良いもの(違うもの)をつくる</h3>

<ul>
<li>10倍良いものをつくろう。競合サービスの延長線上で小さな違いを生むのではなく、顧客にとって価値のある部分で10倍良いものをつくるために大胆にかける。</li>
<li><strong>10倍良いものは、既存サービスとは比較されず、全く違うものと認識される。そうなると「〇〇するなら、△△の一択だ」と選ばれるようになる。</strong>（例：PayPalは小切手送付より10倍速い、Amazonは既存書店より10倍種類が多い、iPadは既存タブレットより10倍良いデザイン）</li>
<li>競合企業を真似ても、それは過去のものであり、お客様にとって類似サービスが増えるだけ。「未来から逆算し、10倍良いもの」を生み出すことに集中しよう。<strong>見るべきは競合企業や目立つリリースではなく、お客様と未来である。</strong></li>
<li><em>“競合サービスは、過去のものであり、私たちがつくるのは10年後の未来です。” 『商品はつくるな 市場をつくれ』</em></li>
</ul>

<h3>ミッションを中心に据え、「良い人」から降りる</h3>

<ul>
<li>All for Mission。すべてはミッションを達成するために行う。<strong>遠慮や保身によって必要な意思決定を遅らせない。</strong>言い難いことも誠実に伝え、必要な変化を求めていく。</li>
<li>時には人としての直感に反する決断を迫られる、タフな状況も起こり得る。それでも、<strong>罪悪感を抱きながらも、それでも前に進む。</strong></li>
<li>例えば、利用者のいるサービスをクローズする判断、あるスタッフにとって厳しい制度を導入する判断、当人の熱量に応えられない人事的な異動判断など。<strong>ミッション達成のために必要な課題から、逃げずに向き合う。</strong></li>
<li>もしも「嫌われたくない」などを理由に、ミッション中心の意思決定から逸れた時、それは自分の信念に誠実ではなくなる瞬間だ。その時から、「どうしてこの仕事をしているのか…？」と疑心暗鬼となり、情熱は損なわれ、疲労だけが残り、最終的にバーンアウトするだろう。</li>
</ul>

<p>参考：<a href="https://kudoshu07.com/2026/03/14/goodperson"><em>『良い人から降りる』</em></a>、<a href="https://kudoshu07.com/2025/01/04/2024germination"><em>「一緒に死ぬでもいい」（2024/02）- 『2025年は「太い幹」をつくる（2024振り返り）』</em></a></p>

<h3>健康第一。メンタル不調を無視しない。</h3>

<ul>
<li>“Hello Vegan!”な社会を実現するには少なくとも数十年、その先の世界平和には数百年かかるだろう。<strong>僕たちの戦いは長期戦</strong>だ。とにかく、健康第一。<strong>健康な心身は、事業活動の大前提</strong>だ。</li>
<li>もしも体調不良を感じたら、何よりもまず<strong>体調回復を最優先</strong>とする。1ヶ月無理をして、倒れて以降働けなくなるような<strong>持続性の低い働き方はやめよう</strong>。「がむしゃらに頑張る」のではなく、限られた時間で「正しい方向に努力を向ける」というのが重要。</li>
<li>特に<strong>メンタル不調を無視しない</strong>。スタートアップは一般企業よりも、倒産するリスクは当然高く、また急成長が求められる。そのため、プレッシャーは他の仕事よりも大きく、精神負荷が高い。</li>
<li>特に創業者は、うつ病になる確率が2倍、双極性障害になる確率が10倍、精神科に入院する可能性が「2倍」、自殺願望を持つ人の割合が「2倍」多い。</li>
<li>一般の社会人以上にセルフケアを意識的に行う必要がある。<strong>しっかり寝る、しっかり食べる、しっかり運動する、しっかり休息の時間を取る、しっかり友人やパートナーなど親しい人との時間を取る</strong>。</li>
</ul>

<p>参考：<a href="https://coralcap.co/2021/11/mental-health-for-ceos/">『メンタルヘルスは事業持続性に関わるCEOの重要スキル』</a></p>

<h3>Do it with joy</h3>

<ul>
<li>スタートアップはハードで、責任範囲も業務量も相対的に多い。嬉しいことよりも辛いこともある。<strong>それでもなぜやるか？面白いからだ。</strong></li>
<li>特に創業期のスタートアップにおいて、「これはいける！」という高揚と、「もう無理かも…」という絶望のジェットコースターに乗っているような日々をおくることになる。そのジェットコースターを楽しめるかどうかが分かれ道になる。<strong>乗らされているのではなく、自ら乗って楽しんでいる。</strong>という、達観した視点から自分を見るように心がけたい。</li>
<li>そして、その楽しさをチームで分かち合おう。</li>
<li><em>”do it with joy”  - 「<a href="https://www.yunusjapan.jp/113370.html">ユヌス・ソーシャル・ビジネスの7原則</a>」</em></li>
</ul>

<p>参考：<a href="https://kudoshu07.com/2022/06/01/withjoy">『最近、仕事が楽しくなってきた話。個人と会社の同一視からの脱却。』</a></p>

<h2>“ヴィーガン領域”で戦う</h2>

<h3>ちゃんと儲けよう</h3>

<ul>
<li>ヴィーガン領域をはじめ、ソーシャルビジネスやNPOの界隈には「稼いではいけない」「金=悪」という呪い(外部からの要求)がなぜか存在する。<strong>明確にその悪習慣に”No”を示し、ちゃんと儲けよう。</strong></li>
<li>社会を変えるイノベーションとは、「<strong>経済的な価値を生み出す新しいモノゴト</strong>」である。社会が変わるイノベーションの前提には、経済価値、つまり利益の増加(売上up or コストdown)がある。<strong>ヴィーガンの選択肢を増加させるためには、ヴィーガン事業が儲けることを証明することが必要</strong>だ。</li>
<li>また、多くの利益が出れば、その分ヴィーガン活動家やファームサンクチュアリなどの<strong>寄付で成立する領域への支援もできるようになる</strong>。会社からはもちろん、利益が増加すれば個人の給与も増加し、個人寄付も多くできるようになる。</li>
</ul>

<p>参考：『<a href="https://kudoshu07.com/2026/03/29/innovation">イノベーションについての学び</a>』</p>

<h3>大きな波に乗り、力を蓄える</h3>

<ul>
<li>ヴィーガン業界は未だ成長途上であり、ヴィーガンだけでは大きな事業にはなり得ない。それゆえ「ヴィーガン×〇〇」のように、〇〇という大きな波に乗る必要がある。例えば、日本におけるインバウンド市場や、欧米エリアにおける日本食市場のような、<strong>トレンドの波に”乗る”。まずはそうして会社を強くする。</strong></li>
<li><strong>次に、蓄えた力をもって、第二の波を”つくる”。</strong>Netflixは最初8年間「DVD市場成長」という第一の波に、DVDレンタルサービスで乗った。そこで得たリソースをもって「ストリーミング」という第二の波をリードした。</li>
<li><em>“成功したどの企業も、成長市場の波に乗っている。2回目の大成功を生むために重要なことは、その波が消える前に新たな大波をつくることだ。” ー Jeff Kagen（<a href="https://gibsonbiddle.medium.com/6-the-glee-model-6af740bdf3b1">#8 The GLEe Model</a>）</em></li>
</ul>

<p>参考：<a href="https://kudoshu07.com/2024/02/08/bigwave">『どの「大きな波」に乗るか？（GLEeモデル）』</a></p>

<h3>“No”に揺らがない。そのためのインサイト。</h3>

<ul>
<li>SNSのコメント欄を見て貰えば分かるように、ヴィーガンの活動には、無数の“No”が浴びせられる。時には、その“No”を内在化させたヴィーガンからの意見をいただくこともある。<strong>“No”にブレない。</strong>僕たちのやるべきことは「誰かの意見通りにやること」ではない。「<strong>事業を成長させ、ミッションを達成すること</strong>」だ。</li>
<li>そのためには、“No”に負けない明確なお客様からのインサイトを心に据える必要がある。<strong>的外れな大きな声に「でも実際はそうじゃないんだよなぁ」と心で反論できる根拠を持とう。</strong></li>
<li>一次情報に触れよう。誰かのアドバイスやAIの意見、ネットのデータなどは、参考にはなるが、確信にはならない。</li>
<li>むしろ、<strong>誰でも得られるデータを根拠に事業をしてしまえば、同じような事業が追随して困難を極める</strong>ことになる。まだ誰も知らないが、<strong>自分だけが知っているインサイトを見つけよう。</strong></li>
</ul>

<p><em>参考：<a href="https://kudoshu07.com/2024/07/14/insightdriven">INSIGHT DRIVEN（インサイトからはじめよ）</a></em></p>

<h3>協力する・協力を求める</h3>

<ul>
<li>ヴィーガン業界は特殊な業界だ。ヴィーガンの生活者、事業者、発信者、活動家など、<strong>全員が「目的を共有した仲間」である</strong>ように感じる。ヴィーガンの選択を増やすこと、ヴィーガン業界を盛り上げること、動物や環境問題を解決すること。目的は一緒だ。</li>
<li>そのため、<strong>僕ができる範囲で最大限協力する</strong>ようにしている。事業の相談やクラファンのサポート、情報の社内外へのシェア、少額だが寄付などが多い。</li>
<li>一方で、<strong>困ったときは協力を求める。</strong>クラファンの支援のお願い、情報拡散の協力依頼をはじめ、飲食事業開始前には飲食店で1日体験させていただいたり、忙しい中でも相談の時間を設けていただいたりした。</li>
<li>ヴィーガン業界（特に今いる熱意あるプレイヤーの皆様）は、一人が勝ち残る業界ではなく、<strong>「全員で勝つ」ができる業界</strong>だ。既存の市場を奪い合うのではなく、<strong>成長過程の市場を全員で育てていく段階だから</strong>だ。</li>
</ul>

<h3>賢く(アホみたいに)立ち回る</h3>

<ul>
<li><strong>Veganismは、未だマジョリティには理解され難い思想だ</strong>。以前、交流会で出会った金融業界の偉い人が「うちにもヴィーガンのやつ(従業員)いるけど、面倒くさいんですよ笑」と冗談めかして僕に伝えてきたことがある。彼自身が歩く社会課題なのだが、実際にそういうことがあってしまうのだ。</li>
<li>現実問題、社会はヴィーガンに不寛容だ。それに対して、不満を伝えたり、敵対心を込めて睨みつけたりすることもできる。だが、僕にとってはその方法が最適だとは思わない。</li>
<li>一方で、<strong>マジョリティが理解でき、あるいは応援しやすい領域も存在する。僕たちの掲げる「Hello Vegan!な社会」は、まさにそうだ</strong>。「選択肢がなくて困っている人がいる。だから選択肢を増やす」というのは、多くの人が理解しやすく、協力を得やすい。ヴィーガン業界で事業をするということは、<strong>ヴィーガン以外のステークホルダー(銀行, 投資家, 物件オーナーなど)と上手く付き合うために、賢く(アホみたいに)立ち回る</strong>。</li>
<li>ただし、その先にある”世界平和”に、動物たちを含めている以上、「畜産動物を０にする」という目的は変わらない。ただし、<strong>それを誰に、どこまで、どう伝えるか？という立ち回りを考慮する</strong>。</li>
</ul>

<h2>“飲食事業”をもって戦う</h2>

<h3>One Teamで、大きな成果をあげよう</h3>

<ul>
<li><strong>チームで協力して成果を上げよう</strong>。飲食事業は、スーパープレイヤーがいても、一人でお店を回すことはできない。店長/副店長、各リーダー、キッチンスタッフ、ホールスタッフなど、<strong>「チーム一丸となる」が前提条件</strong>だ。</li>
<li>そのために、<strong>相手を理解することを楽しむマインド</strong>で他者と関わろう。人種も国籍も、時代や育ってきた環境も違う人と、一緒に働ける環境は一言で言えば面白い。なんで自分と違うんだろう？なんでそんなふうに思うんだろう？なんでこういう行動を取るんだろう？と、どんどん探求していこう。</li>
<li>そして、<strong>“指摘よりもサポート”</strong>というスタンスで関わろう。「出来ていないことを指摘する」のではなく、「どうすれば出来るようになるか？」を考えてサポートしよう。<strong>良いフィードバックは、相手の行動を変える。</strong>悪いフィードバックは相手を萎縮させ行動を無くす。良いフィードバックをしよう。</li>
<li>最後に、<strong>サポートしてもらおう</strong>。ノーミス完璧人間などいないし、そうなる必要もない。出来ないこともあるし、上手くいかない日もある。だからチームでやる。困ったことがあったら助けてもらおう、差し出されている手を掴む勇気を持とう。<strong>それがチーム全体でサポートを求めて良いんだ、という風土につながる</strong>。</li>
</ul>

<h3>現場第一。スタッフへの態度が、お客様への態度となる</h3>

<ul>
<li>飲食事業は、接客業だ。スタッフがお客様に直接関わり、料理を提供し、ちょっとした雑談もする。この時にスタッフの接客態度、超簡単にいうと<strong>「笑顔だったか」がお客様の満足度に大きく影響する</strong>。どれだけ美味しい料理でも、無愛想なスタッフがガタンと配膳しては台無しだ。</li>
<li>では、どうすればスタッフが笑顔で働けるか？それはリーダー層(リーダー,責任者,店長)のスタッフへの態度で変わる。<strong>リーダー層がスタッフに笑顔で、誠実な態度で、思いやりを持って関われば、それは自然とスタッフのお客様への態度に反映される</strong>。逆に「おもんない職場で笑って働け」と言われる方が厳しいものがある。</li>
<li>また、リーダー層の態度は何に影響されているか？それは経営の態度である。だから、<strong>経営は店長などのリーダー層をサポートする。リーダー層はスタッフをサポートする。そうすればスタッフは、お客様に笑顔で接することができる</strong>。</li>
<li><em>“組織図の一番上には私たちが奉仕するお客様、真ん中にはスターバックスで働くパートナーたち、そして1番下がリーダー、つまり経営者だ。”　ー『スターバックスを世界一にするために守り続けてきた大切な原則』より</em></li>
</ul>

<h3>最新技術を取り入れる</h3>

<ul>
<li>積極的に最新技術を取り入れよう。外食産業は歴史あるゆえに、「昔のやり方」が良しとされるケースが多い。一度、<strong>「本当にそれはお客様の価値になっているか？」と問い直してみる</strong>ことが重要だ。</li>
<li><strong>新しい技術は、多くを変える。</strong>手洗いから食洗機に、ハンドドリップからコーヒーマシンになり、薪からガスになった。インターネットができれば、集客にはSNSやレビューが重要になった。冷凍技術が高まれば、今まで提供できなかった食材を提供できるようになった。</li>
<li>同様に、ヴィーガン飲食店として、様々な最新技術を取り入れていきたい。AIの活用、厨房設備、ツール、新たなヴィーガン商品など。<strong>「新しいものを取り入れる」ということも、始めたての小さな飲食店にとっての大きな強みになる。守るものがないからこそ、誰よりも早く新しいものを取り入れていきたい。</strong></li>
</ul>

<h3>裾野を広げない</h3>

<ul>
<li>ヴィーガン業界はもちろん、特に飲食店の方とお話ししていると、「いかに裾野を広げるか？」「多くの人に来店いただくか？」という思考の方が多い。おそらく、商圏が半径数キロで限られていた時代は、100%に近い人に来店してもらえるかを考えるのが正しかったのかもしれない。</li>
<li>しかし、特に都市部や観光地でヴィーガン飲食店の場合、<strong>裾野を広げない方が良い</strong>。つまり、しっかりターゲットを絞っても1%でもいれば、十分な来店数を見込むことができる。インバウンド市場で考えると、<strong>商圏は地球規模</strong>と考えても良い。世界中から渋谷に来店してくださっている。</li>
<li>1%に特化して商品や体験設計をデザインすれば、満足度は跳ね上がる。100人が良いと思うものをつくるのではなく、<strong>1人が感動するものをつくろう。</strong></li>
</ul>

<h3>シングルスタンダード</h3>

<ul>
<li>例えば、キッチンリーダーが「きのこは1分焼いてね」と教えた次の日、店長が「焼き時間は2分だよ」と指示を出すとどうなるか？スタッフは困惑し、それを繰り返すと、その日の上司の顔色を伺って<strong>「じゃあ指示された通りにやればいいや」</strong>と、主体性を失う。<strong>「ダブルスタンダード」は、人間から主体性を奪う。</strong></li>
<li>僕たちが目指すは、<strong>いつでも・人によってブレない基準である「シングルスタンダード」</strong>。シングルスタンダードは、<strong>最低限のクオリティやルールとなり、その上で各スタッフが自由に個性を発揮することができる土台</strong>となる。</li>
<li>最低限のブレない基準があることで、各スタッフが自信を持って、調理や接客にプラスアルファで個性を乗せて取り組むことができる。いつも間違っているかを気にしながら働くよりも、シングルスタンダードの上で自由に働く方が楽しく、面白い。</li>
<li>そのためには、スタッフ全員でシングルスタンダードの意識を高く保つ必要がある。マニュアルやレシピを最新版にキープし、人によって指導が違うことがあれば、そのズレを修正する必要がある。</li>
</ul>

<h3>責任と権限をセットで渡す</h3>

<ul>
<li><strong>責任と権限はセットになっていなければ、成立しない。</strong>例えば、「オペレーションの改善責任」に対して、「5,000円の使用権限」がセットで与えられると、改善に必要な諸々を購入することができる。</li>
<li>しかし、もし経費の使用権限がないとどうだろうか？改善責任はあるのに、改善することができない状態になる。これでは、本人も何もできずに歯痒い思いをする。</li>
<li>反対に、権限だけを与え、責任を渡していないとどうなるか？5,000円の使用目的は曖昧になり、「あったら良さそう」なものに使ってしまうかもしれない。権限があるから使っても良いのに、そうしたら店長や経営層からネガティブフィードバックが来るのは最悪だ。</li>
<li>よって、<strong>「責任」と「権限」をセットで渡すことが重要</strong>だ。それが双方にとって、お客様やミッションに向かってお互いに気持ちよく働くための前提である。</li>
</ul>
