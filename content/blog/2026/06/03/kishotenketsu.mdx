0);">では、なぜ日本では結論から話さないのか？※ビジネスシーンなど除く</p><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);">少し調べてみると面白い観点がいくつも出てきました。</p><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);">まとめるとこんな感じ</p><h3 data-text-color="rgb(0, 0, 0)" data-level="3" style="color: rgb(0, 0, 0);">①動詞が最後の文法</h3><ul><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">日本語の文法は、「〜という理由から、反対です」と伝えられる。そのため、短文であっても、まずは背景を伝えるスイッチになるのは分かる気がする。</p></li><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">日本語頭のまま英語を話そうとすると、動詞（agree/want/needなど）を先に言うのに慣れてなくて、ちょっと詰まる。</p></li><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">一方、英語などの場合「I disagree because…」と、最初に意見を表明する必要あり。</p></li></ul><h3 data-text-color="rgb(0, 0, 0)" data-level="3" style="color: rgb(0, 0, 0);">②共通文脈の多さ</h3><ul><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">日本では、同じ言語の単一民族（北海道や沖縄など除く）かつ、鎖国時代が長く、「日本人」の中に共通文脈が多い。</p></li><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">そのため、”察して”伝わることが多く、結論を先に言う必要性が高くない。</p></li><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">ざっくり欧米では言語や宗教など異なる人が多く、共通文脈が少なく、結論を先にハッキリ伝える必要性が高かった。</p></li></ul><h3 data-text-color="rgb(0, 0, 0)" data-level="3" style="color: rgb(0, 0, 0);">③関係維持の重要度の高さ</h3><ul><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">日本では、藩や家制度などのシステムが発達し、周囲との関係性の重要度が高かったという。「村八分」みたいな言葉があるように、ハブられたら死んじゃう社会だったみたい？</p></li><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">一説によると、「水を共有する稲作では、反感を買うと水が止まる」というリスクがあったからではという論もあるらしい。（ゆえにアジアに多いらしい）</p></li><li data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><p class="bn-inline-content">一方、欧米（特にヨーロッパ）では、日本と比較すると、契約的なシステムが早くに発達し、「はっきりと自分の意見やスタンスを伝える」ことの重要性が高かったという。</p></li></ul><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);">まとめると、</p><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><strong>共通文脈があると「なんとなく分かる」ので、結論を先に言う必要がない。</strong></p><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);"><strong>それよりも、サバイブするために関係性維持の重要性の方が高かった。</strong></p><h2 data-text-color="rgb(0, 0, 0)" data-level="2" style="color: rgb(0, 0, 0);">結：「結・起承転結」をインストールしたい</h2><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);">日本人の僕が、ドイツ人と話す時、ほとんど共通文脈がない。</p><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);">その場合は、<strong>やはり「結論から先に話す」が必要になる。</strong>じゃないと😕という顔をさせてしまうのだ。それでは、良いコミュニケーションのデザインではない。</p><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0, 0);">最近では、ビジネスシーンやSNSにおいて<strong>「結・起承転結」のコミュニケーションスタイルが日本でも広まりつつある。</strong></p><p data-text-color="rgb(0, 0, 0)" style="color: rgb(0, 0,
