<div>

<strong>-Vegan-  </strong>

動物を食べることを避け、卵・乳製品などの動物由来の食品の摂取も避ける人。場合によっては、動物を利用した製品（皮革製品など）の使用も避ける。

</div></blockquote>
<div></div>
&nbsp;

これから、定期的にVegan生活をしていると<strong>よくある質問</strong>について、個人的な回答をしていくシリーズを書いていこうと思います！

&nbsp;

今回の質問は、こちら。
<div></div>
<h1><strong>「</strong><strong>みんなにVeganになって欲しいの？」</strong></h1>
&nbsp;

です！

&nbsp;

大学に入学して新歓の際に先輩方にveganになった背景や思いを話して、この質問をされた記憶が鮮明にあります。

&nbsp;

きっと、自分だけの生活の変化で満足なのか、多くの人々に変化を求めているのかが気になる人が多いのだろうと思います。

&nbsp;
<h2>一言で答えます。</h2>
この質問に対して一言で答えるとしたら、こうです。

&nbsp;
<h1><strong>「もちろんです！」</strong></h1>
&nbsp;

です。

&nbsp;

しかし、これは相手に自分の考えを押し付けるのとは違います。

工藤の理想を言うと、もちろんみんながVeganであって欲しい。

&nbsp;

「ゴミ問題やばいし、ビニール袋できれば使って欲しくない」の思いが強いバージョンというと伝わりやすいと思います。

&nbsp;

&nbsp;

しかし、自分の理想、正しいと信じる正義のぶつけ合いになると、また苦しみが生まれることを知っているので、それはしません。

&nbsp;

ただ、自らの意思でそうなることを願っています。

&nbsp;

&nbsp;

&nbsp;

&nbsp;
<h2>工藤的見解を述べます。</h2>
上の回答だけでは誤解を招きそうなので、その回答の背景にある考えを簡単にちょろっと説明します。

&nbsp;

工藤の大きな理想は、<strong>不幸のない世界</strong>です。

不幸とは、<strong>不条理な境遇によって自らの人生を歩む自由のない状態</strong>です。

&nbsp;

例えば、生まれながらにして貧困で自分の人生を選択できないこどもや、戦地で強制的に戦うことになる若者、そして、生まれてから搾取される為だけに肥やされる家畜もまたそのうちの一つです。

&nbsp;

そんな状況下にある存在を一人でも減らしたいし、それを減らそうとする思いやりとそれに基づく行動こそ、不条理を覆す唯一の根本的解決策であると考えています。

&nbsp;

&nbsp;

そして、肉や牛乳や魚を購入するという行動は、不条理を生み出す方向へ働きます。

少しよくわからないかもしれませんが、シンプルな話です。

&nbsp;

例えば、豚肉を買う場合を考えます。

豚は家畜場で育ち、屠殺場で殺され、加工場で加工され、トラックに運ばれ、スーパーに並べられて、消費者が豚肉を購入します。

&nbsp;

消費者が豚肉を購入した際に支払ったお金は、スーパーへ、運搬会社へ、加工場へ、屠殺場へ、家畜場へと繋がっています。

&nbsp;

&nbsp;

つまり、消費者の豚肉を買うという行動も当然豚肉を生産する行程の一つであり、豚にとっての不条理を生み出している大きな要素であるということです。

<img class="alignnone wp-image-602" src="/wp-content/uploads/2018/03/20170729150209_IMG_1863-300x200.jpg" alt="" width="584" height="389" />

&nbsp;

&nbsp;
<h2>まとめ</h2>
<strong>・家畜が、生まれてから生きる場所、食べるもの、死ぬ時間を決められていることは明らかに不条理な境遇である。</strong>

<strong>・それを生み出している要因として消費者の消費行動がある。</strong>

<strong>・工藤は不条理のない社会を作っていきたい。</strong>

&nbsp;

これらのことから、みんなにVeganになることが理想だと考えています。
