実は僕、2016年の11月から<strong>vegan</strong>というライフスタイルを選択しています。

<a href="https://kudoshu07.com/2017/11/01/vegan/" target="_blank" rel="noopener noreferrer" data-blogcard="1">猫</a>

&nbsp;

このブログでも何度かヴィーガンに対する質問へ答えてきましたが、今回の質問もとてもよくあるものです。その質問とは、こちらです。
<h1><strong>「みんなヴィーガンになったら畜産業を営む人はどうなるの？」</strong></h1>
&nbsp;

&nbsp;

ヴィーガンを始めた背景を話したり、畜産業はこんなんで〜などの話をすると、よくされる質問ですね。

<strong>みんながVeganになれば、畜産業に携わる人たちが仕事を失ってしまう</strong>ことに関して、どう思っているのかが気になるのは自然なことなのかもしれません。

&nbsp;

では、質問の回答をしていきたいと思います。

&nbsp;
<h2><strong>一言で回答</strong></h2>
<img class="alignnone wp-image-1594" src="https://kudoshu07.com/wp-content/uploads/2018/03/工藤　お手手-300x200.jpg" alt="" width="977" height="651" />

みんながヴィーガンになったら畜産業を営む人は職を失ってしまうという質問に対して、　一言で答えるとしたらこうです。

&nbsp;
<h1><strong>「仕方ない！」</strong></h1>
&nbsp;

&nbsp;

しかし、この回答は畜産業に関わる人が仕事を失い、お金を稼ぐ手段がなくなり、貧困状態となって、家族が崩壊し路頭に迷っても構わない、と考えているわけではありません。

勘違いする方が多いので、もう一度言います。<strong>畜産業を営む方が不幸になればいいなんて一切考えていません。</strong>

&nbsp;
<h2><strong>畜産業は持続的ではない</strong></h2>
<img class="alignnone wp-image-624" src="https://kudoshu07.com/wp-content/uploads/2018/03/スクリーンショット-2018-03-22-22.27.10-300x165.png" alt="" width="636" height="350" />

やはり上記の一言と、数行では全てを述べることが出来ないので、もう少し詳細を書いていこうと思います。

&nbsp;

「仕方ない」と言う回答の背景には、他者を傷付ける仕事よりも、<strong>他者を幸せにする仕事をして欲しい</strong>と言う工藤の願いと、人類の繁栄を長期的に見たときに、やはり現在の畜産業の形では<strong>環境に不可が大きく持続的ではない</strong>のでより良い仕事へ代替される事が望ましいという考えがあります。

&nbsp;

上の画像でもわかるように、現在の効率化を求めて行き着いた畜産業は、動物に対してのみだけではなく地球環境や僕たち人間にとっても負荷の強いものです。これは<strong>とても持続可能とは言えない</strong>です。

&nbsp;

ですので、その仕事を続けることで毎日何十万、何百万という<strong>多くの動物の犠牲を生んでしまうことや、環境を破壊し続ける事を肯定することは出来ません</strong>。

&nbsp;
<h2><strong>一気に全員ヴィーガンにならない</strong></h2>
<img class="alignnone wp-image-1596" src="https://kudoshu07.com/wp-content/uploads/2018/03/IMG_8494-300x200.jpg" alt="" width="1169" height="779" />

次に<strong>「全人類がVeganとなり畜産業が無くなる」</strong>というこの質問の前提にある考えについて意見します。

&nbsp;

一般的に考えて、<strong>一気に全人類がヴィーガンになる事はありえない</strong>事です。

これから少しずつ動物への思いやりを認識する人が増え、環境への配慮が大切であるという価値観が育成されていくことで、少しずつ少しずつ人口は増えていくことと思います。

&nbsp;

つまり、一気に畜産業を営む人の仕事がなくなる訳ではありません。時代の流れを掴み、自分の仕事を選択することは重要です。これは畜産業をやっているから怠っていいものではないでしょう。

&nbsp;
<h2><strong>産業は移りゆく</strong></h2>
<img class="alignnone wp-image-1595" src="https://kudoshu07.com/wp-content/uploads/2018/03/IMG_8496-300x200.jpg" alt="" width="1157" height="771" />

次は、ヴィーガンや畜産業に限らず、産業についてです。

時代の移り変わりと共に、産業は繁栄と衰退をゆっくりと繰り返し、衰退する産業に務める人は徐々に減少し、繁栄している(もしくは、しそうな)産業へ労働者は移り行くのではないでしょうか。

&nbsp;

例えば、ビデオテープの時代からDVDの時代へのシフト。

この時に「ビデオテープを作っている人たちが仕事を失ったらどうするんだ。DVDだけを買わず、ビデオテープを使うようにしよう」とはならないですよね。

&nbsp;

みんな、故障が少なく、小さくて軽いDVDへと移行していくはずです。現にそうなっていますね。

つまり<strong>人は自分の価値観に沿ってより良いものを選択し、それに伴って産業もより良いものへとシフトしていきます。</strong>

&nbsp;

それと同様に、環境の不可が大きく動物の権利を侵害する畜産業(少なくとも現在の工場畜産の形態)も、<strong>持続的な人類の発展・</strong><strong>動物の権利という観点</strong>から衰退し、新たな産業が繁栄する時代が来る必要があると考えています。

&nbsp;
<h2><strong>まとめ</strong></h2>
まとめると

<strong>・畜産業を営む人も幸せであって欲しい。</strong>

<strong>・でも畜産業は持続可能でない&amp;動物の権利を侵害している。</strong>

<strong>・少しでもより良い次の産業へシフトする必要がある。</strong>

<strong>・だから畜産業を営む人が他の産業へ移ることは必須。</strong>

<strong>・ただ、全人類が一気にいヴィーガンにはならない。</strong>

<strong>・今すぐ畜産業へ携わる人が仕事を失うことはない。</strong>

&nbsp;

これらのことから、工藤はこれからVeganを実践する人口が増加し、畜産業が衰退する傾向になることを、仕方のない自然のことだと考えています。

畜産業を営む方は、他の仕事や生産システムを改善するなどが必須になると思います。これから畜産業に関わりたいと思っている方にも、この記事を読んで考える機会になれば幸いです。
